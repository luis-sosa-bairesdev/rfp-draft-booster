# Documentation

This directory contains technical documentation for the RFP Draft Booster project.

## Purpose

All technical documentation, architecture decisions, API documentation, and technical guides should be stored here following the `document-template.mc` format.

## Structure

```
documentation/
├── README.md                  # This file
├── architecture/              # Architecture decisions and diagrams
├── api/                       # API documentation
├── deployment/                # Deployment guides
└── decisions/                 # Technical decision records
```

## Document Template

Use the provided `document-template.mc` when creating new technical documents.

## Guidelines

- All documents must be in **English**
- Use Markdown format (.md)
- Follow the template structure
- Keep documents up to date with code changes
- Reference related code and requirements
